El contenido de este archivo es:
-----Exploracion_Cluster_Forecast.ipynb :  
Documento de Jupyter que muestra proceso de exploración y manejo de datos para la clusterización y predicción en series de tiempo 
-----Dashboard_app.ipynb y Dashboard_app.py: 
Documentos en Jupyter y python que muestran la elaboración del dashboard ejecutable en nuestra propia computadora, 
con absoluta necesidad de los archivos en la carpeta "Datos"


-----Carpeta "Datos": 
Carpeta que contiene los archivos derivados de Exploracion_Cluster_Forecast.ipynb, necesarios para la ejecución del dashboard


La información misma información desplegable del dashboard generado por "Dashboard_app.py" puede consultarse en:
